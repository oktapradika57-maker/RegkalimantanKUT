# Regional Kalimantan Operational Dashboard

Dashboard web modern satu halaman (SPA) terintegrasi langsung dengan sumber data utama dari Google Spreadsheet menggunakan sinkronisasi API Realtime Client-Side Fetching.

## Cara Menggunakan
1. Buka file `index.html` dan modifikasi variabel `SPREADSHEET_ID` jika menggunakan file spreadsheet baru.
2. Pastikan izin akses Google Spreadsheet Anda disetel ke **"Anyone with the link can view" (Siapa saja pemilik tautan dapat melihat)** agar API browser dapat membaca data.
